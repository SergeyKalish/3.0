## Обзор проекта

**Hockey Tagger 4.0** — десктопное приложение для разметки и анализа видеозаписей хоккейных матчей. Приложение позволяет:
- Просматривать видео с удобной навигацией и перемоткой
- Размечать временные метки (сегменты, паузы, смены, голы, удаления)
- Автоматически обнаруживать события по видео (OCR для графики, шаблоны для футажей)
- Автоматически рассчитывать временные диапазоны (периоды, ЧИИ — чистое игровое время)
- Вести учёт смен игроков и генерировать отчёты

## Технологический стек

- **Python 3.x** — основной язык
- **PyQt5** — GUI-фреймворк
- **OpenCV (cv2)** — обработка видео
- **Pillow (PIL)** — обработка изображений, генерация отчётов
- **pytesseract** — OCR (распознавание текста на видео)
- **numpy** — численные операции
- **beautifulsoup4 + requests** — парсинг протоколов матчей

## Формат файлов проекта

Проекты сохраняются в файлах с расширением **`.hkt`** (JSON-формат). Основная структура:

```json
{
  "version": "4.0",
  "video_path": "C:/.../match.mp4",
  "match": {
    "generic_labels": [...],      // Пользовательские метки
    "calculated_ranges": [...],   // Рассчитанные диапазоны
    "player_shifts": {...},       // Смены игроков (глобальное время)
    "player_shifts_official_timer": {...},  // Смены в официальном времени
    "teams": {...},               // Информация о командах
    "rosters": {...},             // Составы команд
    "events": [...],              // События из протокола
    "match_id": "...",
    "tournament_name": "...",
    "tour_number": "...",
    "match_date": "...",
    "match_time": "...",
    "venue_city": "...",
    "venue_arena": "...",
    "referees": [...]
  }
}
```

## Типы меток (GenericLabel)

- **Сегмент** — начало/конец периода матча
- **Пауза** — остановки игры (внутри периода)
- **Смена** — изменение состава на льду
- **Гол** — забитый гол
- **Удаление** — штраф/удаление игрока
- **Черн.OSD** — автоматически найденная графика (OCR)
- **Черн.FOOTAGE** — автоматически найденный футаж

## Типы рассчитанных диапазонов (CalculatedRange)

- **Всё видео** — полная длительность
- **Сегмент** — периоды матча (Период 1, Период 2, Период 3, Овертайм)
- **ЧИИ** — чистое игровое время (внутри периодов без пауз)
- **Удаление** — временные интервалы удалений
- **game_mode** — режимы игры (5 на 5, 5 на 4 и т.д.)
- **Счёт** — интервалы с определённым счётом

## Ключевые модули и их назначение

### SMART Processor (`modules/smart.py`)

Центральный модуль анализа данных. Класс `SMARTProcessor` содержит методы:

- `process(generic_labels, total_duration_sec)` — основной метод расчёта диапазонов
- `_process_player_shifts(...)` — обработка смен игроков
- `_process_player_shifts_official_timer(...)` — конвертация в официальное время
- `convert_global_to_official_time(...)` — перевод глобального времени в официальное

Рассчитывает:
- Периоды матча (из пар меток "Сегмент")
- Паузы и ЧИИ (чистое игровое время)
- Удаления (из пар меток "Удаление" с одинаковым контекстом)
- Режимы игры (составы на льду на основе удалений)
- Счёт матча (на основе меток "Гол")
- Смены игроков (из меток "Смена")
- Официальное время (с коэффициентами периодов)

### Auto Draft Marker (`modules/auto_draft_marker.py`)

Автоматическая разметка видео:
- **OSG-детекция**: Ищет ключевые слова ("ГОЛ") в заданной области экрана с помощью OCR
- **Footage-детекция**: Ищет повторяющиеся футажи по ключевому кадру-шаблону

Запускается через `run_marker.py`, который настраивает пути к Qt-плагинам и Tesseract.

### Video Player (`ui/video_player_widget.py`)

Кастомный видеоплеер с функциями:
- Перемотка с разными шагами (±5с, ±30с, ±300с)
- Изменение скорости воспроизведения (от -3.0x до +3.0x)
- Режим "схлопывания" — ограничение навигации выбранным диапазоном
- Горячие клавиши (пробел — play/pause, стрелки — перемотка, 1 — поставить метку, 8/9 — прилипание)

### Timeline Widget (`ui/timeline_widget.py`)

Визуальная временная шкала:
- Отображение меток (вертикальные линии)
- Отображение диапазонов (прямоугольники)
- Фильтрация по типам
- Навигация по клику
- Прилипание к ближайшим элементам (клавиши 8/9)

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Пробел | Play/Pause |
| ← → | Перемотка ±0.5с |
| Ctrl + ← → | Перемотка ±5с |
| Shift + ← → | Перемотка ±30с |
| Ctrl + Shift + ← → | Перемотка ±300с |
| + / - | Увеличить/уменьшить скорость |
| 0 | Сбросить скорость до 1.0x |
| ~ (ё) | Реверсировать направление |
| 1 | Поставить метку выбранного типа |
| 8 | Перейти к предыдущему элементу на шкале |
| 9 | Перейти к следующему элементу на шкале |

## Работа с временем

- **Глобальное время** — секунды от начала видео (float)
- **Официальное время** — игровое время с учётом коэффициентов периодов (float)
- Все метки хранят глобальное время; конвертация в официальное через `utils/helpers.py`

Коэффициенты периодов рассчитываются как отношение фактической длительности ЧИИ к целевой (1200 сек = 20 минут для обычных периодов).

```python
{
  "team": "f-team",

# Смена
{
  "players_on_ice": [{"id_fhm": "...", "name": "..."}, ...]
}
```

## Генерация отчётов

Модуль `modules/reports/` создаёт визуальные отчёты "Карта смен" в формате PNG:
- Лист "Матч" — полная статистика по всему матчу
- Листы "Период 1/2/3" — статистика по периодам
- Графическое отображение смен, голов, удалений, вбрасываний
- Таблица с суммарной статистикой по игрокам

Размер листа: A4 или A3 (альбомная ориентация), 300 DPI.

## Расширение функциональности

### Добавление нового типа метки
1. Добавить тип в `UniversalLabelEditor.label_types`
2. Добавить цвет в `TimelineWidget.COLOR_MAPPING` (для отображения)
3. Добавить обработку в `SMARTProcessor.process()` (если нужен расчёт диапазонов)

### Добавление нового отчёта
1. Создать класс в `modules/reports/`
2. Унаследовать от базового класса или использовать `PlayerShiftMapReport` как пример
3. Добавить UI-диалог при необходимости
